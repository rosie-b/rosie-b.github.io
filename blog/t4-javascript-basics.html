JavaScript Basics

How does JavaScript compare to HTML and CSS?

HTML is about the structure of your site and CSS is about the decoration, with the combination of these being somewhat dynamic (e.g. click a button, links to another page; or changing the layout depending on screen size).

JavaScript is about getting into the finer details of creating objects and telling these objects how to behave. This could be in the properties it has (e.g. you have a car, with the properties: red, 4x wheels, 1x horn), perhaps a sound it makes (e.g. when horn button pushed, make X sound) and/or the way that it moves (e.g. when break pushed, decelerate by X). And you can control the way in which various elements interact with eachother.

We use JavaScript across the world to build dynamic, interactive, fun and useful apps and games.

Control flow and loops

In JavaScript, the control flow refers to the order in which the statements are executed. This may include if/else statements, which would mean that blocks of code would be executed only if certain criteria were met. Statements can also be set to run on loops until certain criteria are met. An example of this process in real life could be making porridge, where the steps may be:

  1. Put pot on stove
  2. Put oats in pot
  3. Pour milk on oats
  4. Turn on stove to medium heat
  5. ** Start loop, repeat step and stop once porridge starts to boil ** Stir porridge every minute
  6. Turn stove down to low heat
  7. ** Start loop, repeat for 1 minute ** Stir porridge constantly
  8. Pour porridge into bowl

Accessing data from arrays and objects

Accessing data from arrays is via an index, whereas data from objects is accessed via their properties.

An array is a way of listing data numerically, so when accessing the information you would reference the index number (Note: The index is zero-based, meaning that the first item in the list will be 0, the second will be 1 etc).

An object is defined by a range of properties, consisting of the name of the property and then the value assigned to this (e.g. "colour": red;). So when accessing this data, you would reference the name of the property ("colour").

Functions

Functions are procedures in JavaScript, using statements to make calculations or perform certain tasks. This can be completed once, multiple times and/or be executed based on certain criteria being met.

These are useful because you are setting up a set of instructions that can be executed throughout your code and this means that you don't need to write the instructions out in full every single time.
